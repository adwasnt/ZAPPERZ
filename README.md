# ZAPPERZ
A simple space invaders recreation that I made in my Computer Graphics class.

# How to Play
All of the instructions are in the `zapgame_instructions.hmtl`

# Controls
Gun Movement:
  - Move Left: LeftArrow, a, j
  - Move Right: RightArrow, d, l
  - Fire Laser: f
